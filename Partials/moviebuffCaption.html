Movie Buff was completed as a group project (team of 3) during my time at Nashville Software School. This app allows users to create an account, find movies (via the IMDB movie api), add movies to their own collection, search their movie collection, mark movies as 'watched', rate 'watched' movies on scale of 1-10, and filter 'watched' movies according to their rating.

Built using JavaScript, jQuery, Require.js, Bootstrap, Firebase, Sass, Lodash, HTML5, and CSS3.

Deployed via DigitalOcean using NGINX.
